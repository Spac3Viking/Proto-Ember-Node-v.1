# Philosophy Cartridge — Core Notes

## On Knowledge and Signal

Knowledge is not information. Information is raw signal — text, numbers, patterns.
Knowledge is signal that has been understood, verified against experience, and integrated
into a coherent framework of understanding.

The distinction matters because:
- Information can be copied without comprehension
- Knowledge must be earned through engagement
- An archive of information is a database
- An archive of knowledge is a library that can think

## The Examined Life

Socrates famously held that the unexamined life is not worth living.

For a knowledge engine, the parallel is: unexamined data is not worth storing.

Before material enters Hearth, it should be examined. What does it claim? What is its
source? How does it relate to what is already known? What would it change if accepted?

The Threshold → Workshop → Hearth path enforces this discipline structurally.

---

## Epistemic Humility

A local knowledge engine should be epistemically humble:
- Prefer remembered, verified local sources over speculation
- Acknowledge when local knowledge is insufficient
- Never invent answers that are not grounded in available signal
- Make the absence of knowledge visible: "That signal has not reached this hearth."

---

## Recursive Refinement

Philosophy is itself a recursive practice: ideas are refined through dialogue, tested
against counter-arguments, revised, and refined again.

The Ember Node models this structurally. Material passes through the node's rooms
multiple times: arriving raw, being refined, being remembered, being retrieved and
combined with new material, being refined again.

Each cycle adds resolution.
