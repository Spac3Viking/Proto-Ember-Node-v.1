No Local AI Detected

Ember Node works best with a local AI model host. The recommended option is Ollama.

Setup steps for Ollama
  1. Visit ollama.com and download Ollama for your OS.
  2. Install and run: ollama serve
  3. Pull a model: ollama pull gemma3:4b
  4. Return here and click ↺ Scan to detect Ollama.

Ember Node will never auto-install tools. All setup is manual and under your control.
