# Journals Cartridge

This cartridge holds personal journal entries, logs, and reflections indexed for AI retrieval.

## Purpose
Use this cartridge to store timestamped notes, daily logs, and personal reflections that the Heart Assistant can reference during conversations.

## Format
Each journal entry should be a separate Markdown file named by date, e.g. `2026-03-12.md`.

## Notes
Your entries remain entirely local — they are never sent to external services.
